# Tactile Hide and Seek

This is the repository that contains source code for the [Tactile Hide and Seek project page](https://tactile-hide-and-seek.github.io).

## About

**Tactile Hide and Seek: Bimanual Object Blind Search and Retrieval via Tactile-Only Feedback**

Xiangyu Fu, Hao Xing, Simon Armleder, Wenlan Shen, Fengyi Wang, Gordon Cheng

Institute for Cognitive Systems, Technical University of Munich

We present a tactile-only method for bimanual object search and retrieval in vision-denied environments using custom tactile skin sensors. The system employs a two-phase classification process and achieves 91.1% object classification accuracy and 61.4% success rate in real-world experiments.
